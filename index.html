Git 筆記本

git基本語法

git + add 檔名 儲存該檔案
add . 全部儲存
commit -m '' 增加標籤(可多行註解)
commit 在預設編輯器編輯commit
log 可以看head所在branch的所有commit
reflog 全部的commit過程都可起看到(總覽的概念)
status 有沒有檔案未add跟commit
checkout 轉換到某個commit或branch
switch 轉換branch
branch+名字(自行輸入) 新增一個branch，該branch名字為輸入字樣
*如果先checkout回過去又add新結點，再用branch雖會開啟新branch，但是head仍在原本的分支上
